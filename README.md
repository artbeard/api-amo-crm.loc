# Тестовая задача для работы с api amoCRM

Перед началом работы для авторизации нужно вписать apiKey в файл .env в корне проекта.
При создании сущности имя можно не вводить, просто нажмите Ок.

## Installation
```bash
$ npm install
```

## Running the app
```bash
# watch mode
$ npm run start:dev
```
